# AI QA Automation Framework

## Overview
This project is an AI-powered QA Automation Framework designed to test APIs and scalable systems using modern testing practices.

## Tech Stack
- Python
- Pytest
- Requests
- API Testing

## Features
- API Automation Framework
- Modular Design (Client + Test separation)
- Scalable structure for future UI & AI testing
- Clean and reusable code
